C Library for ML in FlexACC

发表于2025-07-24|更新于2025-07-24|Computer Architecture
|阅读量:

This library’s functions are all in C language. Please follow the guidance of interface definition. Remember that as to accelerate the computation and realize efficient SIMD operations, all value should be store in memory cache using vector-like store (such as actvec_t, vec_t, mat_t and etc.)

Convolutional Operations

conv2d

In the part, we are going to introduce conv2d operation that may suitable for image related machine learning workloads. The funtion has inferface like:

  • out_base: the output memory start pointer for output matrix

  • in_base: the input memory start pointer for input matrix

  • weight_base: the weight start pointer

  • bias_vec: the bias add memory start pointer

  • IN_H, IN_W, IN_C: input data’s heigh, weight and channel numbers

  • OUT_C, K: define the output channel and convolutional kernel size

  • STRIDE, PAD: define the step size of moving the convolutional window across the input and padding extra pixels around the border of the input image

  • mul_o, shf_o: define the optional quantization operation

  • apply_relu: judge whether to add relu layer behind convolution computing.

  • IN_H, IN_W, C: input data’s heigh, weight and channel numbers
